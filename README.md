# Covarmaille（カバーメイル）

**Remote Work Robot Platform - ソーシャルビジネスプロジェクト**

## ✨ キャッチフレーズ
**「機械、意思、重ねて」**

> *Covar（Cover + Armor）+ maille（仏：鎧）*  
> 人の意思と機械が重なるとき、身体的制約を越えた働き方が生まれる。  
> 職種の選択肢に不自由があってはならない。

## 🎯 ミッション
ALS患者や身体的制約により外出困難な方々に、ロボット技術を通じた新しい働き方を提供し、社会参加と経済的自立を実現する。

## 💡 コンセプト
**「誰もが、どこからでも、自分らしく働ける社会へ」**

遠隔操作可能な汎用ロボットプラットフォームと人材マッチングサービスを組み合わせ、身体的制約のある方々の就労機会を創出するソーシャルビジネス。

### コアバリュー：職種選択の自由
**「接客だけではない。あなたの意思で、あなたの職種を」**

既存のテレプレゼンスロボットが接客・受付業務に限定されるのに対し、Covarmailleはモジュラー設計により多様な職種への対応を実現。接客、作業支援、クリエイティブ、専門業務まで、働く人の意思とスキルに合わせた職種選択を可能にします。

## 🤖 プロダクト概要

### 開発するロボット
**職種選択の自由を実現するモジュラーロボットシステム**

> OriHimeは接客・受付に特化。Covarmailleは接客から作業支援、クリエイティブ、専門業務まで、  
> 働く人の意思とスキルに合わせた多様な職種への対応を目指します。

1. **Type A: 接客・受付特化型**
   - テレプレゼンス機能（映像・音声通信）
   - 表情表現・ジェスチャー機能
   - 移動機能（自律 + 遠隔操作）
   - タッチパネル連携

2. **作業支援ロボット**
   - 遠隔操作可能なロボットアーム
   - 物品運搬・軽作業対応
   - カフェ・飲食店での作業支援
   - オフィスでの書類整理等

3. **統合プラットフォーム**
   - モジュラー設計（用途に応じたカスタマイズ）
   - クラウドベースの操作システム
   - 低遅延通信技術
   - AIアシスト機能

## 💼 ビジネスモデル

### 収益の柱
1. **人材紹介料 / 派遣料**
   - 企業から月額または時間単位で課金
   - マッチング手数料

2. **ロボットレンタル / 販売**
   - 月額レンタルプラン
   - 買取オプション
   - メンテナンスサービス

## 🎯 ターゲット

### 働き手側
- ALS患者
- 重度身体障がい者
- 在宅介護者
- その他外出困難な方々

### 企業側
- 接客業（ホテル、施設受付、観光案内）
- 飲食業（カフェ、レストラン、配膳サポート）
- 小売業（店舗接客、在庫管理）
- オフィス（受付、データ入力、軽作業）
- クリエイティブ（デザイン、ライティング、編集）
- 専門業務（プログラミング、翻訳、コンサルティング）
- 教育（オンライン講師、学習サポート）
- 製造業（検品、軽作業支援）
