# Covarmaille（カバーメイル）ビジネスプラン詳細

## 1. エグゼクティブサマリー

### 会社名の由来
**Covarmaille（カバーメイル）**
- **Covar**: Cover（補う）+ Armor（鎧）
- **maille**: フランス語で「鎧」「メッシュ」
- **意味**: 身体的制約を補い、人を守る技術の鎧。柔軟性と強度を兼ね備えた、すべての人に働く機会を提供する存在。

### キャッチフレーズ
**「機械、意思、重ねて」**

人の意思と機械の技術が重なるとき、身体的制約を越えた新しい働き方が生まれる。機械は人の意思を実現するパートナーであり、人は機械を通じて自分の意思を世界に届ける。

### 事業概要
身体的制約により外出困難な方々に、遠隔操作可能な汎用ロボットと就労機会を提供するソーシャルビジネス。ロボット開発と人材マッチングサポートを活用。

### 解決する社会課題
- ALS患者や重度身体障がい者の就労機会の欠如
- 在宅介護者の経済的困窮
- 企業の人手不足（特にサービス業）
- ダイバーシティ＆インクルージョンの実現

インスピレーションやデータ元:
- https://www.soumu.go.jp/main_sosiki/joho_tsusin/top/tokusyu.html (遠隔操作ロボットと介護連携)
- https://www.jil.go.jp/kokunai/statistics/index.html (障害者雇用統計データ)

### 競合優位性

#### コアコンセプト：職種選択の自由
**「ロボットで行う職種には、基本的に選択肢の不自由があってはならない」**

身体的制約のある方々が働く場合、その選択肢が「接客・受付」に限定されるべきではありません。人はそれぞれ異なるスキル、経験、情熱を持っています。Covarmailleはモジュラー設計とAI支援技術により、接客から作業支援、クリエイティブ、専門業務まで、多様な職種への対応を実現します。

#### OriHimeとの差別化

| 比較項目 | OriHime | Covarmaille |
|---------|---------|-------------|
| **対応職種** | 接客・受付に限定 | **接客から作業、専門業務まで幅広く対応** |
| **汎用性** | 会話・プレゼンス中心 | **作業支援機能統合** |
| **モジュール性** | 固定デザイン | **業務別カスタマイズ可能** |
| **作業機能** | なし | **ロボットアーム搭載可能** |
| **AI支援** | 限定的 | **自律機能・操作補助** |
| **操作IF** | タブレット中心 | **多様な入力デバイス対応** |
| **プラットフォーム** | ロボット販売中心 | **人材マッチング統合** |

**重要：** Covarmailleの強みは、働く人の「意思」と「スキル」に合わせて職種を選べること。  
デザイナーはデザインを、プログラマーはプログラミングを、接客の得意な人は接客を。
ロボットはその意思を実現するパートナーとして機能します。
