# ビジネスプラン詳細

## 1. エグゼクティブサマリー

### 事業概要
身体的制約により外出困難な方々に、遠隔操作可能な汎用ロボットと就労機会を提供するソーシャルビジネス。ロボット開発と人材マッチングサービスを統合した、新しい働き方のプラットフォームを構築する。

### 解決する社会課題
- ALS患者や重度身体障がい者の就労機会の欠如
- 在宅介護者の経済的困窮
- 企業の人手不足（特にサービス業）
- ダイバーシティ＆インクルージョンの実現

### 競合優位性
- **OriHimeとの差別化**
  - より汎用性の高いモジュラー設計
  - 作業支援機能の統合
  - AIアシスト機能
  - 業務特化型カスタマイズ
  - 総合的な人材マッチングプラットフォーム

## 2. 市場機会

### ターゲット市場規模

#### 働き手側
- ALS患者：日本国内約1万人
- 重度身体障がい者（1-2級）：約110万人
- 在宅介護者：約600万人
- **潜在的ユーザー数：数十万人規模**

#### 企業側
- サービス業の人手不足：慢性的な課題
- ダイバーシティ雇用のニーズ増加
- テレワーク/リモートワーク技術への投資意欲

### 市場トレンド
- テレプレゼンスロボット市場の成長（年率20%+予測）
- ダイバーシティ経営の重要性増大
- 障がい者雇用促進法による法定雇用率の引き上げ
- ロボット技術の進化とコスト低下

## 3. プロダクト戦略

### 製品ラインナップ

#### Type A: 接客・受付特化型
**用途**: ホテル受付、施設案内、イベントスタッフ
- テレプレゼンスディスプレイ（顔表示）
- 音声通信システム
- 自律移動 + 遠隔操作
- ジェスチャー機能
- タッチパネル連携
- **想定価格**: 
  - レンタル: 10-15万円/月
  - 販売: 200-300万円

#### Type B: カフェ・軽作業支援型
**用途**: カフェスタッフ、軽作業、物品運搬
- 遠隔操作ロボットアーム
- トレイ運搬機能
- 簡易作業（物を運ぶ、置く）
- テレプレゼンス機能併用
- **想定価格**:
  - レンタル: 15-20万円/月
  - 販売: 300-500万円

#### Type C: オールインワン型
**用途**: 複合的な業務が必要な現場
- Type A + Type B の統合
- モジュラー交換可能
- AIアシスト機能
- **想定価格**:
  - レンタル: 25-35万円/月
  - 販売: 500-800万円

### 技術的差別化要素

1. **モジュラー設計**
   - 業務に応じてパーツ交換可能
   - アップグレード容易

2. **AIアシスト**
   - 自律ナビゲーション
   - 障害物回避
   - 操作補助（初心者でも扱いやすい）
   - 音声認識・自動応答

3. **低遅延通信**
   - 5G対応
   - エッジコンピューティング活用
   - 快適な操作感

4. **操作インターフェース**
   - 視線入力対応（ALS患者向け）
   - ゲームコントローラー対応
   - 音声コマンド対応
   - カスタマイズ可能なUI

## 4. ビジネスモデル

### 収益構造

#### 1. ロボットレンタル収益
- **月額レンタルプラン**
  - Type A: 10-15万円/月
  - Type B: 15-20万円/月
  - Type C: 25-35万円/月
- **メンテナンス込み**
- **初期費用**: 設置費用 10-20万円

#### 2. ロボット販売収益
- Type A: 200-300万円
- Type B: 300-500万円
- Type C: 500-800万円
- **保守契約**: 2-5万円/月

#### 3. 人材マッチング収益
- **企業から徴収**
  - 月額派遣料: 時給1,500-2,000円 × 稼働時間
  - マッチング手数料: 派遣料の20-30%
  
- **働き手へ支払い**
  - 時給: 1,000-1,400円
  
- **マージン**: 300-600円/時間

#### 4. プラットフォーム利用料
- クラウドシステム利用料: 5,000-10,000円/月/ユーザー
- データ通信費: 実費 + マージン

### 収益シミュレーション（3年後目標）

**前提条件**:
- ロボット配置台数: 200台
- 稼働率: 70%
- 平均稼働時間: 6時間/日 × 20日/月

**月次収益予測**:
1. レンタル収益: 200台 × 20万円 = 4,000万円
2. 人材マッチング収益: 200台 × 70% × 120時間 × 400円 = 672万円
3. プラットフォーム利用料: 200ユーザー × 8,000円 = 160万円

**月次合計**: 約4,832万円
**年間売上**: 約5.8億円

## 5. マーケティング戦略

### Phase 1: 認知獲得（最初の6ヶ月）
- **ターゲット**: ALS協会、障がい者支援団体
- **手法**:
  - 患者会・支援団体との連携
  - メディア露出（社会的意義を訴求）
  - デモンストレーション
  - パイロットプログラム

### Phase 2: 企業開拓（6-18ヶ月）
- **ターゲット**: ホテル、カフェ、オフィスビル
- **手法**:
  - 導入実績の訴求
  - ROI（投資対効果）の可視化
  - CSR/ESG観点からの提案
  - 無料トライアル（1ヶ月）

### Phase 3: スケール（18ヶ月以降）
- **ターゲット**: 全国展開
- **手法**:
  - 代理店網の構築
  - オンラインマーケティング
  - 導入事例集の展開

## 6. パートナーシップ戦略

### 必要なパートナー

1. **医療・福祉団体**
   - ALS協会
   - 障がい者支援NPO
   - 介護関連団体
   - → ユーザー獲得、ニーズ把握

2. **ロボット・技術パートナー**
   - ロボットメーカー
   - 通信事業者（5G）
   - AI開発企業
   - → 技術開発、製造

3. **人材・雇用パートナー**
   - ハローワーク
   - 障がい者就労支援機関
   - 人材派遣会社
   - → マッチング、研修

4. **導入先企業**
   - ホテルチェーン
   - カフェチェーン
   - オフィスビル管理会社
   - → パイロット導入、実績作り

5. **金融機関**
   - ソーシャルインパクト投資ファンド
   - 地方銀行
   - → 資金調達

## 9. リスクと対策

### 技術リスク
- **リスク**: ロボット開発の遅延、性能不足
- **対策**: 既存技術の活用、段階的開発、プロトタイプでの早期検証

### 市場リスク
- **リスク**: 企業の導入ハードルが高い
- **対策**: 無料トライアル、ROI可視化、CSR観点での訴求

### 規制リスク
- **リスク**: ロボット使用に関する規制
- **対策**: 事前の法規制調査、行政との連携

### 競合リスク
- **リスク**: OriHimeや他社の追随
- **対策**: 先行者利益の確保、特許出願、プラットフォーム戦略

## 10. 社会的インパクト指標（KPI）

### 定量指標
- 就労機会創出数（人）
- 総労働時間（時間）
- 収入向上額（円）
- 企業の人手不足解消率（%）

### 定性指標
- ユーザーのQOL向上
- 社会参加の実感
- 企業のダイバーシティ推進
