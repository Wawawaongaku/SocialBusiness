# 市場分析

## 1. 市場規模

### テレプレゼンスロボット市場

#### グローバル市場
```
2024年: 約300億円
2030年予測: 約1,200億円
CAGR: 約20-25%

成長要因:
- リモートワークの定着
- ロボット技術の進化
- 5G通信の普及
- 人手不足の深刻化
```

#### 日本市場
```
2024年: 約30-40億円
2030年予測: 約150-200億円
CAGR: 約25-30%

成長要因:
- 超高齢社会での労働力不足
- ダイバーシティ推進
- 障がい者雇用促進
- DX推進
```

### 障がい者就労支援市場

#### 市場規模
```
障がい者雇用市場:
- 民間企業の障がい者雇用数: 約61万人（2023年）
- 法定雇用率: 2.3% → 2.5%（2024年）→ 2.7%（2026年）
- 雇用義務企業: 約10万社

就労支援サービス市場:
- 市場規模: 約3,000億円
- 成長率: 年率5-10%
```

### ターゲット顧客規模

#### 働き手側（供給側）

##### ALS患者
```
日本国内患者数: 約10,000人
年間新規発症: 約2,000人

就労希望率: 20-30%（推定）
→ 潜在ユーザー: 2,000-3,000人

特徴:
- 高学歴・高スキル層が多い
- コミュニケーション能力維持（初期）
- 視線入力等での操作可能
```

##### 重度身体障がい者
```
身体障がい者手帳保有者: 約436万人
うち1-2級（重度）: 約110万人

在宅・就労希望者: 10-20%（推定）
→ 潜在ユーザー: 10-20万人

特徴:
- 多様な障がい状態
- 個別のカスタマイズニーズ
- 就労経験・スキルの多様性
```

##### 在宅介護者
```
在宅介護者数: 約600万人
うち就労困難者: 約100-200万人

就労希望者: 30-50%（推定）
→ 潜在ユーザー: 30-100万人

特徴:
- 時間的制約
- フルタイム困難
- 柔軟な働き方ニーズ
```

**総潜在市場（供給側）: 40-120万人**

#### 企業側（需要側）

##### 接客・サービス業
```
ホテル・宿泊業:
- 施設数: 約8万施設
- 慢性的な人手不足
- 受付業務の自動化ニーズ

飲食業:
- 店舗数: 約60万店舗
- 特にカフェ・喫茶: 約7万店舗
- 人件費圧迫、採用困難

小売業:
- 店舗数: 約130万店舗
- レジ・接客の省人化ニーズ
```

##### オフィス・事業所
```
オフィスビル:
- 大規模ビル: 約5,000棟
- 受付・案内業務

企業・事業所:
- 従業員規模50人以上: 約5万社
- 障がい者雇用義務企業: 約10万社
- CSR/ESG推進企業: 増加傾向
```

**総潜在市場（需要側）: 
- 初期ターゲット（大手・先進企業）: 5,000-10,000社
- 中期ターゲット: 5-10万社**

## 2. 競合分析

### 直接競合

#### OriHime（オリィ研究所）
```
強み:
- 市場の先駆者（ブランド認知度高い）
- 分身ロボットカフェ「DAWN」実績
- 豊富な導入事例
- メディア露出多数
- 障がい者コミュニティとの強い繋がり

弱み:
- 機能が会話・プレゼンス中心
- 作業機能なし
- カスタマイズ性低い
- 価格帯: 約50-200万円

市場シェア:
- 日本の分身ロボット市場で最大手
- 累計導入台数: 数百台規模（推定）
```

#### Telepresence Robot各種
```
Double Robotics:
- iPad搭載型移動ロボット
- 価格: 約30-50万円
- 主に企業向け

Beam（Suitable Technologies）:
- 高性能テレプレゼンス
- 価格: 約200万円
- 法人向け

その他:
- Kubi（卓上型）
- Padbot
- 中国製低価格品
```

## 3. 顧客ニーズ分析

### 働き手側のニーズ

#### Primary Needs（必須ニーズ）
```
1. 就労機会の獲得
   - 収入確保
   - 社会参加
   - 自己実現

2. 身体的制約の克服
   - 外出不要
   - 自宅からの就労
   - 体調に応じた柔軟な働き方

3. 操作の容易さ
   - 自身の障がい状況に応じた操作方法
   - 疲労が少ない
   - 直感的なUI
```

#### Secondary Needs（期待ニーズ）
```
1. キャリア形成
   - スキルアップ機会
   - 多様な業務経験
   - 正当な評価

2. コミュニティ
   - 同じ境遇の仲間
   - 孤立感の解消
   - 情報交換

3. サポート体制
   - 技術的サポート
   - 就労相談
   - トラブル対応
```

### 企業側のニーズ

#### Primary Needs（必須ニーズ）
```
1. 人手不足の解消
   - 採用難の業務を補填
   - 安定的な人材確保
   - コスト効率

2. 業務の効率化
   - 定型業務の自動化
   - 生産性向上
   - オペレーション改善

3. 投資対効果（ROI）
   - 人件費との比較
   - 初期投資の回収期間
   - ランニングコスト
```

#### Secondary Needs（期待ニーズ）
```
1. 企業イメージ向上
   - CSR活動としての訴求
   - ESG経営の実践
   - メディア露出効果

2. ダイバーシティ推進
   - 法定雇用率の充足
   - インクルーシブな職場
   - 障がい者雇用の実績

3. イノベーション
   - 最新技術の導入
   - 先進的な取り組み
   - 業界での差別化
```

## 4. 市場参入戦略

### ターゲット市場の優先順位

#### Phase 1: Early Adopter（0-12ヶ月）
```
ターゲット企業:
- CSR/ESGに積極的な大企業
- テクノロジー先進企業
- 社会課題解決志向の企業

想定業種:
- ホテル・宿泊業（受付）
- カフェ・飲食（先進的な店舗）
- IT企業（オフィス受付）

目標:
- 10-20社の導入
- 成功事例の創出
- メディア露出
```

#### Phase 2: Early Majority（12-36ヶ月）
```
ターゲット企業:
- 中堅企業
- チェーン店舗
- 人手不足が深刻な業種

想定業種:
- ホテルチェーン
- カフェチェーン
- 商業施設

目標:
- 100-200社の導入
- ROI実証データの蓄積
- 代理店網の構築
```

#### Phase 3: Late Majority（36ヶ月以降）
```
ターゲット企業:
- 中小企業
- 地方企業
- 慎重な企業

施策:
- 低価格モデルの投入
- 補助金・助成金の活用
- 代理店経由の拡販
```

### 差別化戦略

#### 1. 製品差別化
```
- 作業機能の統合（OriHimeとの差別化）
- モジュラー設計による汎用性
- AIアシストによる操作性向上
- 多様な入力デバイス対応
```

#### 2. サービス差別化
```
- 人材マッチングの統合
- 充実したサポート体制
- 継続的な改善・アップデート
- コミュニティ形成
```

#### 3. 価格戦略
```
- レンタルモデルでの初期投資抑制
- 段階的な機能追加（アップセル）
- ボリュームディスカウント
- 補助金活用のサポート
```

## 5. 市場機会

### 追い風要因

#### 1. 社会トレンド
```
✓ 働き方改革の推進
✓ ダイバーシティ&インクルージョン
✓ ESG経営の重視
✓ DX推進
✓ 人手不足の深刻化
```

#### 2. 技術トレンド
```
✓ 5G通信の普及
✓ AI技術の進化
✓ ロボット技術の成熟
✓ クラウドサービスの普及
✓ デバイスコストの低下
```

#### 3. 政策・制度
```
✓ 障がい者法定雇用率の引き上げ
✓ テレワーク推進政策
✓ ロボット導入補助金
✓ 働き方改革関連法
```

### リスク要因

#### 1. 市場リスク
```
⚠ 景気悪化による企業の投資抑制
⚠ 既存サービスの機能向上
⚠ 大手企業の参入
```

#### 2. 技術リスク
```
⚠ 技術的課題（遅延、操作性等）
⚠ セキュリティ・プライバシー懸念
⚠ 通信インフラの制約
```

#### 3. 社会的リスク
```
⚠ ロボットへの抵抗感
⚠ 雇用代替への懸念
⚠ 規制強化の可能性
```
