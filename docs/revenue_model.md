# 収益モデル詳細

## 1. 収益構造の全体像

### ビジネスモデルの特徴
```
ハイブリッド収益モデル:
① ロボットレンタル/販売（ハードウェア）
② 人材マッチング（プラットフォーム）
③ サービス・サポート（継続課金）

特徴:
✓ 複数の収益源によるリスク分散
✓ 継続的な収益（ストック収益）
✓ スケーラビリティ
```

## 2. 収益源別詳細

### 収益源①: ロボットレンタル

#### 料金体系
```
Type A（接客・受付特化型）:
□ 月額レンタル料: 10-15万円/月
  - 基本プラン: 10万円
  - プレミアムプラン: 15万円（AIアシスト強化）

Type B（カフェ・軽作業支援型）:
□ 月額レンタル料: 15-20万円/月
  - 基本プラン: 15万円
  - プレミアムプラン: 20万円

Type C（オールインワン型）:
□ 月額レンタル料: 25-35万円/月
  - 基本プラン: 25万円
  - プレミアムプラン: 35万円
```

#### 初期費用
```
□ 設置費用: 10-20万円
  - 現地調査・設置
  - 初期設定
  - スタッフトレーニング（2-3時間）

□ デポジット: レンタル料の2-3ヶ月分
  - 解約時返金
```

#### 契約期間
```
□ 最低契約期間: 12ヶ月
□ 自動更新: 1年毎
□ 中途解約: 違約金（残期間の50%）
```

#### 収益試算（Year 3目標）
```
配置台数: 100台
内訳:
- Type A: 50台 × 12万円 = 600万円/月
- Type B: 30台 × 17万円 = 510万円/月
- Type C: 20台 × 30万円 = 600万円/月

月次収益: 1,710万円
年間収益: 2億520万円
```

### 収益源②: ロボット販売

#### 販売価格
```
Type A: 200-300万円
Type B: 300-500万円
Type C: 500-800万円
```

#### 販売モデル
```
□ 一括購入
  - 初期導入費用込み
  - 1年間の保証付き

□ 分割払い（リース）
  - 3-5年リース
  - リース会社と提携
```

#### 保守契約（購入時必須）
```
□ ベーシック: 2-3万円/月
  - リモート監視
  - ソフトウェア更新
  - 電話サポート

□ プレミアム: 5-7万円/月
  - ベーシック内容 +
  - 訪問メンテナンス（年2回）
  - 優先サポート
  - パーツ交換無料
```

#### 収益試算（Year 3目標）
```
販売台数: 20台/年
平均単価: 400万円

年間販売収益: 8,000万円
保守契約収益: 20台 × 4万円 × 12ヶ月 = 960万円/年
```

### 収益源③: 人材マッチング

#### 料金構造
```
企業側課金:
□ 派遣料: 時給1,500-2,000円
  - 業務内容・時間帯により変動
  - 深夜・休日は+25%

□ マッチング手数料: 派遣料の20-30%
  - 長期契約で割引
  - ボリュームディスカウント
```

#### 働き手への支払い
```
□ 時給: 1,000-1,400円
  - 経験・スキルにより変動
  - 評価制度による昇給

□ インセンティブ:
  - 顧客満足度ボーナス
  - 長時間稼働ボーナス
  - スキルアップボーナス
```

#### マージン構造
```
例）時給1,800円の場合:
- 企業支払い: 1,800円/時間
- マッチング手数料（25%）: 450円
- 実質派遣単価: 1,350円/時間
- 働き手支払い: 1,200円/時間
- 純マージン: 150円/時間

+ マッチング手数料: 450円/時間
= 合計マージン: 600円/時間
```

#### 収益試算（Year 3目標）
```
稼働ロボット: 100台
稼働率: 70%
平均稼働時間: 6時間/日 × 20日/月 = 120時間/月

稼働台数: 100台 × 70% = 70台
総稼働時間: 70台 × 120時間 = 8,400時間/月
マージン: 8,400時間 × 600円 = 504万円/月

年間収益: 6,048万円
```

## 5. 価格戦略

### 価格設定の考え方

#### バリューベース価格設定
```
企業が得る価値:
□ 人件費削減効果:
  - 従業員雇用: 月給20-25万円 + 社会保険
  - 本サービス: レンタル15万円 + 派遣料12万円（月60時間稼働）
  - 合計: 27万円
  
  → 従来の人件費と同等かやや高い
  → しかし柔軟性・CSR価値で正当化

□ 採用コスト削減:
  - 求人広告費: 10-50万円
  - 採用工数
  - 早期離職リスク

□ 社会的価値:
  - CSR/ESGスコア向上
  - ブランドイメージ向上
  - メディア露出
```

#### 競合比較価格設定
```
□ OriHime:
  - 本体価格: 約50-200万円
  - 運用コスト別途
  
→ 本サービスはレンタルモデルで初期負担軽減

□ 人材派遣:
  - 時給1,500-2,000円
  - 派遣会社マージン: 30-40%
  
→ 本サービスは同等の価格帯

□ 正規雇用:
  - 月給20-25万円 + 社会保険
  - 採用・教育コスト
  - 離職リスク
  
→ 本サービスは柔軟性で優位
```

### 価格戦略オプション

#### 1. ペネトレーション価格戦略（市場浸透）
```
初期: 低価格設定で市場シェア獲得
- レンタル: 10万円/月（通常15万円）
- 初年度のみ特別価格
- 早期導入インセンティブ

メリット:
✓ 早期の市場シェア獲得
✓ 実績・データ蓄積
✓ ネットワーク効果

デメリット:
⚠ 初期の収益性低い
⚠ 価格引き上げが困難
```

#### 2. スキミング価格戦略（高価格）
```
初期: 高価格設定でアーリーアダプター向け
- レンタル: 20万円/月
- プレミアム・限定感の演出

メリット:
✓ 高い初期収益
✓ ブランド価値向上
✓ 段階的値下げ可能

デメリット:
⚠ 市場浸透が遅い
⚠ 競合参入の余地
```

#### 3. フリーミアム戦略
```
基本: プラットフォームは無料
課金: 付加価値サービス

例:
- マッチングプラットフォーム: 無料
- ロボットレンタル: 有料
- プレミアム機能: 追加課金

メリット:
✓ ユーザー獲得が容易
✓ ネットワーク効果
✓ アップセル機会

デメリット:
⚠ 収益化まで時間
⚠ 無料ユーザーのコスト
```

### 推奨戦略

**ハイブリッド戦略**
```
Phase 1（Year 1-2）: ペネトレーション
- 特別価格でアーリーアダプター獲得
- 実績・データ蓄積

Phase 2（Year 2-3）: 価格正常化
- 標準価格への移行
- 早期導入者への優遇継続

Phase 3（Year 3以降）: 価値ベース価格
- セグメント別価格設定
- 付加価値サービスでの差別化
```
