# プロダクト仕様書

## 1. プロダクトコンセプト

### OriHimeとの差別化ポイント

| 比較項目 | OriHime | 本プロダクト |
|---------|---------|-------------|
| **汎用性** | 会話・プレゼンス中心 | 作業支援機能統合 |
| **モジュール性** | 固定デザイン | 業務別カスタマイズ可能 |
| **作業機能** | なし | ロボットアーム搭載可能 |
| **AI支援** | 限定的 | 自律機能・操作補助 |
| **操作IF** | タブレット中心 | 多様な入力デバイス対応 |
| **プラットフォーム** | ロボット販売中心 | 人材マッチング統合 |

## 2. ハードウェア仕様

### Type A: 接客・受付特化型

#### 基本スペック
```
サイズ: 
- 高さ: 120-150cm（可変）
- 幅: 40cm
- 奥行: 40cm
- 重量: 15-20kg

ディスプレイ:
- 顔表示用タッチスクリーン: 10-12インチ
- 解像度: 1920×1080以上
- 情報表示用サブディスプレイ: 7インチ

移動機能:
- 自律移動: LiDAR + カメラベース
- 最高速度: 1.5m/s
- 連続稼働時間: 8-10時間
- 充電時間: 2-3時間
```

#### 通信機能
```
カメラ:
- メインカメラ: 4K 60fps
- 広角カメラ: 180度視野
- 深度カメラ: ToF方式

マイク:
- 指向性マイクアレイ（4個）
- ノイズキャンセリング機能
- エコーキャンセリング

スピーカー:
- ステレオスピーカー
- 360度音声出力

通信:
- 5G対応
- Wi-Fi 6対応
- Bluetooth 5.0
- 有線LAN（バックアップ）
```

#### インタラクション機能
```
ジェスチャー:
- 頭部: 3軸（うなずき、首振り、傾け）
- 腕: 簡易的な挙手・指差しジェスチャー

表情表現:
- ディスプレイ上でのアニメーション表情
- 音声に合わせた口の動き
- 感情表現（喜び、悲しみ等）

センサー:
- 人感センサー
- 障害物検知センサー
- 転倒防止センサー
```

### Type B: カフェ・軽作業支援型

#### 基本スペック
```
サイズ:
- 高さ: 130-160cm（可変）
- 幅: 50cm
- 奥行: 50cm
- 重量: 25-35kg

ロボットアーム:
- 自由度: 6軸
- 可搬重量: 2-3kg
- リーチ: 60cm
- 精度: ±1mm
```

#### 作業機能
```
グリッパー:
- 電動グリッパー
- 把持力: 調整可能（10-100N）
- 交換可能（トレイホルダー等）

トレイ運搬:
- トレイサイズ: 40cm × 30cm
- 耐荷重: 3kg
- 安定化機構付き

作業範囲:
- テーブル高さ対応: 60-100cm
- 棚への配置: 50-180cm
```

#### その他機能
```
- Type Aの通信機能全て搭載
- 作業エリア用補助カメラ（ハンドカメラ）
- 力覚フィードバック（オプション）
```

### Type C: オールインワン型

Type A + Type B の全機能統合
- モジュール交換により用途切替可能
- 統合制御システム
- より高度なAI機能

## 4. 通信・セキュリティ

### 通信仕様
```
プロトコル:
- WebRTC（映像・音声）
- WebSocket（制御信号）
- HTTPS/REST API（データ通信）

遅延目標:
- 映像遅延: 100ms以下
- 操作遅延: 50ms以下

帯域要件:
- アップリンク: 5-10Mbps
- ダウンリンク: 10-20Mbps
```

### セキュリティ
```
通信:
- エンドツーエンド暗号化
- TLS 1.3
- VPN対応

認証:
- 多要素認証
- 生体認証対応（オプション）
- セッション管理

アクセス制御:
- ロールベースアクセス制御
- 操作ログ記録
- 異常アクセス検知
```
