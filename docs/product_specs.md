# プロダクト仕様書

## 1. プロダクトコンセプト

### コアコンセプト：職種選択の自由
**「機械、意思、重ねて」**

ロボットで行う職種には、基本的に選択肢の不自由があってはならない。人の意思と機械の技術が重なることで、接客から作業支援、クリエイティブ、専門業務まで、働く人のスキルと意思に合わせた多様な働き方を実現する。

**職種対応の範囲：**
- 接客・受付（ホテル、施設、観光案内）
- 作業支援（カフェ、飲食、軽作業）
- クリエイティブ（デザイン、編集、ライティング）
- 専門業務（プログラミング、翻訳、コンサルティング）
- 教育（オンライン講師、学習サポート）
- 製造業支援（検品、組立て補助）

### 製造戦略：オープンソース × 3Dプリンター

#### 基本方針
**「重労働でない限り、オープンソースベースの3Dプリンター製ロボットを自社チューニング」**

```
メリット:
✓ 製造コスト70-80%削減
✓ カスタマイズの自由度が高い
✓ 迅速な改良・プロトタイピング
✓ パーツの即時製造・修理が可能
✓ 在庫リスク軽減（必要な時に製造）
✓ 地産地消モデルが可能
✓ オープンソースコミュニティの活用
```

#### 活用するオープンソースプロジェクト

| プロジェクト | 特徴 | 活用部分 |
|----------------|------|----------|
| **InMoov** | 等身大ヒューマノイド、3Dプリント可能 | Type Aの本体フレーム |
| **Poppy Project** | モジュラー式、教育向け | モジュール設計参考 |
| **Niryo One/Ned** | 6軸ロボットアーム、オープンソース | Type Bのアーム部分 |
| **Open Manipulator** | ROS対応アーム | ソフトウェア連携 |
| **OpenCV AI Kit** | AIカメラ | 物体認識・ナビゲーション |

#### 製造方式の使い分け

| 用途 | 製造方式 | 理由 |
|------|---------|------|
| 接客・受付（軽負荷） | 3Dプリンター製 | コスト最小化 |
| カフェ・軽作業（2kg以下） | 3Dプリンター + 既製アーム | コストと機能のバランス |
| 重量物運搬（5kg以上） | 金属製・既製品ベース | 安全性・耐久性確保 |
| 製造業支援 | 産業用ロボット改造 | 精度・耐久性必須 |

### OriHimeとの差別化ポイント

| 比較項目 | OriHime | 本プロダクト |
|---------|---------|-------------|
| **対応職種** | 接客・受付に限定 | **接客から専門業務まで幅広く対応** |
| **製造方式** | 専用金型・工場製造 | **オープンソース3Dプリンター製** |
| **製造コスト** | 高（専用設計） | **低（70-80%削減）** |
| **汎用性** | 会話・プレゼンス中心 | 作業支援機能統合 |
| **モジュール性** | 固定デザイン | 業務別カスタマイズ可能 |
| **作業機能** | なし | ロボットアーム搭載可能 |
| **AI支援** | 限定的 | 自律機能・操作補助 |
| **操作IF** | タブレット中心 | 多様な入力デバイス対応 |
| **プラットフォーム** | ロボット販売中心 | 人材マッチング統合 |
| **価格帯** | 50-200万円 | **30-150万円（大幅コスト削減）** |
| **修理・カスタマイズ** | メーカー依存 | **自社で即時対応可能** |

**重要：** Covarmailleの最大の差別化は「職種選択の自由」。  
接客業務だけでなく、デザイナー、プログラマー、ライターなど、  
働く人のスキルと意思に合わせた多様な職種での働き方を可能にします。

## 2. ハードウェア仕様

### Type A: 接客・受付特化型（3Dプリンター製）

#### 製造方式
```
ベースプロジェクト: InMoov（カスタマイズ版）
素材: PLA/PETG樹脂
製造方式: FDM 3Dプリンター
製造時間: 約50-80時間/台（パーツ合計）
材料費: 2-3万円
```

#### 基本スペック
```
サイズ: 
- 高さ: 120-150cm（可変）
- 幅: 40cm
- 奥行: 40cm
- 重量: 8-12kg（3Dプリンター製で軽量化）

ディスプレイ:
- 顔表示用タッチスクリーン: 10-12インチ
- 解像度: 1920×1080以上
- 情報表示用サブディスプレイ: 7インチ

移動機能:
- 自律移動: LiDAR + カメラベース
- 最高速度: 1.5m/s
- 連続稼働時間: 8-10時間
- 充電時間: 2-3時間
```

#### 通信機能
```
カメラ:
- メインカメラ: 4K 60fps
- 広角カメラ: 180度視野
- 深度カメラ: ToF方式

マイク:
- 指向性マイクアレイ（4個）
- ノイズキャンセリング機能
- エコーキャンセリング

スピーカー:
- ステレオスピーカー
- 360度音声出力

通信:
- 5G対応
- Wi-Fi 6対応
- Bluetooth 5.0
- 有線LAN（バックアップ）
```

#### インタラクション機能
```
ジェスチャー:
- 頭部: 3軸（うなずき、首振り、傾け）
- 腕: 簡易的な挙手・指差しジェスチャー

表情表現:
- ディスプレイ上でのアニメーション表情
- 音声に合わせた口の動き
- 感情表現（喜び、悲しみ等）

センサー:
- 人感センサー
- 障害物検知センサー
- 転倒防止センサー
```

### Type B: カフェ・軽作業支援型（3Dプリンター製 + 既製アーム）

#### 製造方式
```
本体: 3Dプリンター製（InMoovベース）
アーム: Niryo Ned2/One（オープンソース）または既製品
ハイブリッド製造: 耐久性必要部分のみ既製品
製造時間: 約80-120時間/台
材料費: 5-8万円
```

#### 基本スペック
```
サイズ:
- 高さ: 130-160cm（可変）
- 幅: 50cm
- 奥行: 50cm
- 重量: 15-20kg（アーム含む）

ロボットアーム:
- 自由度: 6軸
- 可搬重量: 2-3kg
- リーチ: 60cm
- 精度: ±1mm
```

#### 作業機能
```
グリッパー:
- 電動グリッパー
- 把持力: 調整可能（10-100N）
- 交換可能（トレイホルダー等）

トレイ運搬:
- トレイサイズ: 40cm × 30cm
- 耐荷重: 3kg
- 安定化機構付き

作業範囲:
- テーブル高さ対応: 60-100cm
- 棚への配置: 50-180cm
```

#### その他機能
```
- Type Aの通信機能全て搭載
- 作業エリア用補助カメラ（ハンドカメラ）
- 力覚フィードバック（オプション）
```

### Type C: オールインワン型

Type A + Type B の全機能統合
- モジュール交換により用途切替可能
- 統合制御システム
- より高度なAI機能
